# Stepper 计步器
> 常见的计步器

## 基本用法

```vue
<vu-stepper v-model="num"></vu-stepper>
```

## 参数
| 参数        | 说明             | 类型    | 可选值 | 默认值 |
|-------------|----------------|---------|--------|--------|
| v-model     | 计步器的值       | number  | -      |        |
| min | 最小值| number| -      |  1      |
| max       | 最大值 | number | -| 9999  |
| type       | 样式类型 | string | normal, simple| normal  |
