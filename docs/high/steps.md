#  Steps 步骤组件
> 常见的步骤组件


## 基本用法

```vue
<vu-steps :list="list" :status="status"></vu-steps>
```


## 参数
| 参数      | 说明           | 类型    | 可选值              | 默认值   |
|-----------|--------------|---------|---------------------|----------|
| list      | 步骤列表       | array   | -                   |          |
| status      | 状态值，到第几步       | number | -                   |     |
| dir      | 排列方向 | string  |vertical, horizontal                     | vertical |
